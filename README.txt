===============================================================
DATASET: Global Material Areas (COG)
===============================================================

OVERVIEW:
This Cloud Optimized GeoTIFF (COG) contains estimated spatial
mining areas for specific materials derived from mining cluster data.

CITATION & SOURCE:
Article: Maus, V., et al. (2025). An open database on global coal and metal mine production. SSRN. https://dx.doi.org/10.2139/ssrn.5408302

TECHNICAL SPECIFICATIONS:
- Resolution:      0.5 degrees
- Coordinate Sys:  +proj=longlat +datum=WGS84 +no_defs
- Data Type:       INT2U (16-bit Unsigned Integer)
- NoData Value:    65535

SCALING & UNITS:
IMPORTANT: Values are stored as integers to minimize file size.
Units are Square Kilometers (km2) scaled by a factor of 10.

Formula: Actual Area (km2) = Pixel Value / 10
